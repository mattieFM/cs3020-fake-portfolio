Riot Games API UI Wrapper Documentation

Overview

This program provides a C# wrapper for interacting with the Riot Games API. It allows you to fetch summoner data, game status, match history, and more. The application also includes a simple Windows Forms UI to search for summoner information.

Classes

1. Data

Handles API interactions and provides methods for fetching and parsing data from Riot Games' endpoints.

  • Enums: queue_types - Defines game queue types.
  • Methods:
    • GenericRequestToAPI - Sends a generic API request.
    • GetSummonerPuuidByNameAndTaglineJobj - Fetches a summoner's PUUID using their name and tagline.
    • GetSummonerAccountFromPuuid - Fetches account details using a PUUID.
    • GetSummonerPuuidByNameAndTagline - Retrieves PUUID as a string.
    • GetSummonerDataFromPuuid - Fetches detailed summoner data using a PUUID.
    • GetLolStatus - Gets the current game server status.
    • GetQueueEntiresFromSummonerId - Retrieves queue entries for a summoner by their ID.
    • GetMatchIdsByPuuid - Retrieves match IDs for a summoner using their PUUID.

2. CoolStuff

Acts as a controller for the program, connecting the API logic in Data with the user interface in Form1.

  • Methods:
    • test - A test function to demonstrate API calls.
    • getSummonerData - Fetches summoner data and opens the Summoner Information page.
    • Get_API_Key - Retrieves the API key from the form.
    • setTempLabel - Updates a temporary label on the UI.

3. Form1

The main form of the application. Includes text fields and buttons for user interaction.

  • Methods:
    • getTempLabel - Returns the temporary label for displaying test data.
    • getInputtedName - Retrieves the summoner name input from the UI.
    • getInputtedTag - Retrieves the tagline input from the UI.
    • Get_API_Key - Retrieves the API key input from the UI.
    • button1_Click - Handles the search button click event.

4. SummonerDataPage

Displays detailed summoner data, including level, name, and additional stats. Acts as a child form in the main application.

  • UI Components:
    • textBox1 - Displays the summoner name.
    • textBox3 - Displays the summoner's tagline.
    • textBox4 - Displays additional details (e.g., level).

Usage

To use this program:

  1. Input the summoner name and tagline in the respective fields.
  2. Enter your Riot Games API key.
  3. Click the search button to fetch and display summoner data.

Dependencies

  • Newtonsoft.Json - For parsing JSON responses from the Riot API.
  • System.Net.Http - For making HTTP requests.

Other Documentation

This document provides an overview and usage instructions for the classes and forms in the project.

1. QueueRankDisplay Class

This partial class defines a form used to display queue rank information for a player.

Key Methods:

  • InitializeComponent: Sets up the form’s layout and properties.
  • Dispose: Releases resources used by the form.

Usage:

        
    QueueRankDisplay queueRankDisplay = new QueueRankDisplay(queueData);
    queueRankDisplay.AddThisPageAsDirectChild(parentForm, x, y);

2. SummonerDataPage Class

A form designed to display summoner data retrieved from the League of Legends API.

Key Methods:

  • SummonerDataPage(string puuid, Data data): Constructor initializing the page with player data.
  • SetUpPage: An asynchronous method to fetch and display summoner information.
  • GetSummonerNameBox: Retrieves the TextBox containing the summoner’s name.
  • GetSummonerLvlBox: Retrieves the TextBox displaying the summoner’s level.

Usage:

        
    SummonerDataPage summonerPage = new SummonerDataPage(puuid, data);
    summonerPage.Show();

3. ChildForm Class

This helper class extends `Form` to provide additional functionality for embedding forms as children of other forms.

Key Methods:

  • AddThisPageAsMdiChild: Embeds the form as an MDI (Multiple Document Interface) child.
  • AddThisPageAsDirectChild: Embeds the form as a child form within a panel in the parent form.
  • AddThisPageAsDialog: Displays the form as a modal dialog.

Usage:

        
    ChildForm childForm = new ChildForm();
    childForm.AddThisPageAsDirectChild(parentForm, x, y);

Note: Ensure the parent form has a panel to host child forms when using AddThisPageAsDirectChild.

4. BorderPanel Class

A custom panel with a configurable border color and width.

Key Properties:

  • BorderColor: Defines the color of the border (default: black).
  • BorderWidth: Specifies the width of the border (default: 2).

Key Method:

  • OnPaint: Overrides the default paint behavior to draw a border around the panel.

Usage:

        
    BorderPanel borderPanel = new BorderPanel
    {
        BorderColor = Color.Red,
        BorderWidth = 3,
        Dock = DockStyle.Fill
    };
    parentForm.Controls.Add(borderPanel);

5. Data Class

A helper class to fetch data from the League of Legends API.

Key Methods:

  • GetSummonerDataFromPuuid: Retrieves summoner data using a player’s PUUID.
  • GetSummonerAccountFromPuuid: Fetches account details from a PUUID.
  • GetLolStatus: Returns the current status of League of Legends servers.
  • GetQueueEntriesFromSummonerId: Retrieves ranked queue entries for a summoner.

Usage:

        
    JObject summonerData = await data.GetSummonerDataFromPuuid(puuid);
