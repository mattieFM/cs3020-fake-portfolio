Learnings and Takeaways from CS 3020

This course provided a strong foundation in C#. Here are some of my key takeaways of what the class attempted to teach:

  • Object-Oriented Programming: Reinforced the importance of encapsulation, inheritance, and polymorphism in writing reusable code.
  • WinForms: Gained practical experience in building user interfaces using Windows Forms.
  • API Integration: Learned how to connect to external APIs and process data dynamically.
  • Data prep: Linq -- c# data processing, effectively just week sql with c# flavoring
  • Error Handling: Improved my understanding of exception handling in C# for creating robust applications.

This course also provided valuable insight into structuring applications and handling unexpected limitations, like API restrictions. Though the class itself was structured admittedly poorly and the content covered seems incredibly entry level and not representative of a 3000 level class, the professor did do a good job teaching.
